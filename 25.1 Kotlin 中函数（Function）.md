# 8. 递归函数

```kotlin
fun factorial(n: Int): Int {
    if (n == 1) return 1
    return n * factorial(n - 1)
}
```

调用：

```kotlin
factorial(5)
```

结果：

```text
120
```

---

# 9. 尾递归函数（tailrec）

Kotlin优化递归。

```kotlin
tailrec fun factorial(
    n: Int,
    result: Long = 1
): Long {
    return if (n == 1)
        result
    else
        factorial(
            n - 1,
            result * n
        )
}
```

避免栈溢出。

==**let作用域函数**==
`let` 是 Kotlin 中最常用的**作用域函数（Scope Function）**之一，在 Android 开发中出现频率非常高。

# let 的定义

```kotlin
inline fun <T, R> T.let(
    block: (T) -> R
): R
```

可以理解为：

```kotlin
对象.let {
    // 使用对象
}
```

其中：

- `it` 代表当前对象
    
- `let` 返回 Lambda 最后一行的结果
    

---

# 1. 基本用法

```kotlin
val name = "Tom"

name.let {
    println(it)
}
```

输出：

```text
Tom
```

这里：

```kotlin
it == name
```

---

# 2. 非空判断（最常用）

Android 面试必问。

Java：

```java
if(user != null){
    System.out.println(user.getName());
}
```

Kotlin：

```kotlin
user?.let {
    println(it.name)
}
```

等价于：

```kotlin
if (user != null) {
    println(user.name)
}
```

---

# 3. 避免重复变量名

不使用 let：

```kotlin
val user = getUser()

if (user != null) {
    println(user.name)
    println(user.age)
    println(user.address)
}
```

使用 let：

```kotlin
getUser()?.let {
    println(it.name)
    println(it.age)
    println(it.address)
}
```

---

# 4. 返回处理结果

```kotlin
val result = "123".let {
    it.toInt()
}
```

结果：

```kotlin
result = 123
```

因为 let 返回最后一行：

```kotlin
it.toInt()
```

# 9. let 与 run 区别

### let

使用 `it`

```kotlin
user?.let {
    println(it.name)
}
```

---

### run

使用 `this`

```kotlin
user?.run {
    println(name)
}
```

---

示例：

```kotlin
data class User(
    val name: String,
    val age: Int
)
```

```kotlin
val user = User("Tom", 18)

user.let {
    println(it.name)
}

user.run {
    println(name)
}
```

效果相同。

---

# 10. let 与 apply 区别

### let

返回 Lambda 结果

```kotlin
val result = "abc".let {
    it.length
}

println(result)
```

输出：

```text
3
```

---

### apply

返回对象本身

```kotlin
val user = User()
    .apply {
        name = "Tom"
        age = 18
    }
```

返回：

```kotlin
User
```

---

# Kotlin 五大作用域函数对比

|函数|对象引用|返回值|
|---|---|---|
|let|it|Lambda结果|
|run|this|Lambda结果|
|with|this|Lambda结果|
|apply|this|对象本身|
|also|it|对象本身|

# Android 面试标准回答

当面试官问：

> Kotlin let 有什么作用？

可以回答：

> let 是 Kotlin 的作用域函数，本质是扩展函数。通常用于空安全处理、链式调用和临时作用域。对象会作为 `it` 传入 Lambda，`let` 返回 Lambda 最后一行的结果。在 Android 开发中最常见的场景是 `?.let{}` 做非空判断。

例如：

```kotlin
user?.let {
    tvName.text = it.name
}
```

这样可以避免大量的 `if (user != null)` 判断。
