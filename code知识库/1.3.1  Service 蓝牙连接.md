```gradle
}
```

### 1.2 权限声明

```xml

<!-- Android 12+ -->

<uses-permission android:name="android.permission.BLUETOOTH_CONNECT" />
```

### 1.3 最小化使用示例

```kotlin
// Activity 中
class MainActivity : AppCompatActivity() {
    private var bluetoothService: BluetoothService? = null
    private var isBound = false
    
    // 1. Service 连接
    private val serviceConnection = object : ServiceConnection {
        override fun onServiceConnected(name: ComponentName?, service: IBinder?) {
            bluetoothService = (service as BluetoothService.LocalBinder).getService()
            isBound = true
        }
        override fun onServiceDisconnected(name: ComponentName?) { isBound = false }
    }
    
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        
        // 2. 启动并绑定 Service
        startService(Intent(this, BluetoothService::class.java))
        bindService(Intent(this, BluetoothService::class.java), 
                   serviceConnection, Context.BIND_AUTO_CREATE)
        
        // 3. 注册广播接收器
        val filter = IntentFilter().apply {
            addAction(BluetoothService.ACTION_CONNECTION_STATE_CHANGED)
            addAction(BluetoothService.ACTION_DATA_RECEIVED)
            addAction(BluetoothService.ACTION_CONNECTION_FAILED)
        }
        LocalBroadcastManager.getInstance(this).registerReceiver(broadcastReceiver, filter)
    }
    
    // 4. 广播接收器
    private val broadcastReceiver = object : BroadcastReceiver() {
        override fun onReceive(context: Context?, intent: Intent?) {
            when (intent?.action) {
                BluetoothService.ACTION_CONNECTION_STATE_CHANGED -> {
                    val connected = intent.getBooleanExtra(
                        BluetoothService.EXTRA_STATE, false)
                    Log.d("TAG", "连接状态: $connected")
                }
                BluetoothService.ACTION_DATA_RECEIVED -> {
                    val data = intent.getStringExtra(BluetoothService.EXTRA_DATA)
                    Log.d("TAG", "收到数据: $data")
                }
                BluetoothService.ACTION_CONNECTION_FAILED -> {
                    val error = intent.getStringExtra(BluetoothService.EXTRA_ERROR)
                    Log.e("TAG", "连接失败: $error")
                }
            }
        }
    }
    
    // 5. 用户操作
    fun connect(device: BluetoothDevice) = bluetoothService?.connectDevice(device)
    fun send(text: String) = bluetoothService?.sendText(text)
    fun disconnect() = bluetoothService?.disconnect()
    
    override fun onDestroy() {
        super.onDestroy()
        unbindService(serviceConnection)
        LocalBroadcastManager.getInstance(this).unregisterReceiver(broadcastReceiver)
    }
}
```

## 二、架构设计

### 2.1 整体分层

```
┌─────────────────────────────────────────────────────┐
│                    UI 层 (Activity)                  │
│  - 权限请求 | 界面更新 | 用户交互                    │
└─────────────────┬─────────────────────────────────┘
                  │
         ┌────────┴────────┐
         │ bindService     │ sendBroadcast
         ↓                 ↑
┌─────────────────────────────────────────────────────┐
│               Service 层 (BluetoothService)          │
│  - 生命周期 | 连接管理 | I/O 管理 | 广播通知         │
└─────────────────┬─────────────────────────────────┘
                  │
         ┌────────┴────────┐
         ↓                 ↓
    ┌────────────┐    ┌──────────────┐
    │ 连接线程   │    │ 读取线程     │
    │ (Connect)  │    │ (Reader)     │
    └────────────┘    └──────────────┘
         │                 │
         └────────┬────────┘
                  ↓
        ┌─────────────────────┐
        │  Socket/Stream      │
        │  (硬件蓝牙模块)      │
        └─────────────────────┘
```

### 2.2 五大模块

#### Service 的 5 个模块

|模块|职责|关键方法|
|---|---|---|
|**LifecycleModule**|生命周期管理|onCreate, onBind, onDestroy|
|**StateModule**|状态管理|isConnected, connectedDevice|
|**ConnectionModule**|连接管理|connectDevice, disconnect, cleanup|
|**CommunicationModule**|数据收发|sendData, sendText, startReadThread|
|**BroadcastModule**|广播通知|sendBroadcast (3 种事件)|

#### Activity 的 7 个模块

|模块|职责|关键方法|
|---|---|---|
|**LifecycleModule**|Activity 生命周期|onCreate, onDestroy|
|**PermissionModule**|权限检查请求|checkAndRequestPermissions|
|**ServiceBindingModule**|Service 绑定|bindBluetoothService|
|**BroadcastModule**|广播接收|registerBroadcastReceiver|
|**UserInteractionModule**|用户操作|connectToDevice, sendData, disconnectDevice|
|**CallbackModule**|事件回调|onConnectionStateChanged, onDataReceived|
|**UIUpdateModule**|UI 更新|updateConnectionUI, updateDataUI|

### 2.3 线程模型

```
主线程
├─ Activity 生命周期 ✓
├─ UI 更新 ✓
├─ Service.bindService() ✓
└─ ❌ 连接、I/O 操作 → ANR!

后台线程
├─ Socket.connect() ✓ (可能 10+ 秒)
├─ InputStream.read() ✓ (永久阻塞)
└─ 周期性任务（心跳、重连）✓
```

**关键原则**

- ✗ 不能在主线程执行阻塞操作 → 会 ANR
- ✓ 所有网络 I/O 都在后台线程
- ✓ UI 更新在主线程（用 lifecycleScope.launch）

### 2.4 权限体系

```
Android 11 及以下                    Android 12+
├─ BLUETOOTH                        ├─ BLUETOOTH
├─ BLUETOOTH_ADMIN                  ├─ BLUETOOTH_ADMIN  
├─ ACCESS_FINE_LOCATION             ├─ BLUETOOTH_SCAN (新)
└─                                  ├─ BLUETOOTH_CONNECT (新)
                                    └─ ACCESS_FINE_LOCATION
```

**权限检查流程**

```kotlin
if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.S) {
    // Android 12+：请求新权限
    add(Manifest.permission.BLUETOOTH_SCAN)

```


## 三、完整代码

### 3.1 BluetoothService (Service)

```kotlin
package com.example.bluetooth

import android.app.Service
import android.bluetooth.BluetoothAdapter
import android.bluetooth.BluetoothDevice
import android.bluetooth.BluetoothManager
import android.bluetooth.BluetoothSocket
import android.content.Context
import android.content.Intent
import android.os.Binder
import android.os.Build
import android.os.IBinder
import android.util.Log
import androidx.localbroadcastmanager.content.LocalBroadcastManager
import java.io.IOException
import java.io.InputStream
import java.io.OutputStream
import java.util.UUID

/**
 * BluetoothService - 蓝牙连接的核心 Service
 * 
 * 职责：
 * - 管理蓝牙连接生命周期
 * - 处理数据收发
 * - 通过广播通知 Activity 状态变化
 */
class BluetoothService : Service() {

    // ═══════════════════════════════════════════════════
    // 1. 生命周期模块
    // ═══════════════════════════════════════════════════
    
    private val binder = LocalBinder()
    private var bluetoothAdapter: BluetoothAdapter? = null
    
    companion object {
        private const val TAG = "BluetoothService"
        const val UUID_SERIAL_PORT = "00001101-0000-1000-8000-00805F9B34FB"
        
        // 广播 action
        const val ACTION_CONNECTION_STATE_CHANGED = "com.example.bluetooth.CONNECTION_STATE_CHANGED"
        const val ACTION_DATA_RECEIVED = "com.example.bluetooth.DATA_RECEIVED"
        const val ACTION_CONNECTION_FAILED = "com.example.bluetooth.CONNECTION_FAILED"
        
        // 广播 extra
        const val EXTRA_STATE = "state"
        const val EXTRA_DATA = "data"
        const val EXTRA_ERROR = "error"
    }
    
    inner class LocalBinder : Binder() {
        fun getService(): BluetoothService = this@BluetoothService
    }
    
    override fun onCreate() {
        super.onCreate()
        Log.d(TAG, "✓ Service 创建")
        val bluetoothManager = getSystemService(Context.BLUETOOTH_SERVICE) as BluetoothManager
        bluetoothAdapter = bluetoothManager.adapter
    }
    
    override fun onStartCommand(intent: Intent?, flags: Int, startId: Int): Int {
        Log.d(TAG, "✓ Service 启动")
        return START_STICKY
    }
    
    override fun onBind(intent: Intent?): IBinder? {
        Log.d(TAG, "✓ Activity 已绑定")
        return binder
    }
    
    override fun onDestroy() {
        super.onDestroy()
        Log.d(TAG, "✓ Service 销毁")
        disconnect()
    }
    
    // ═══════════════════════════════════════════════════
    // 2. 状态模块
    // ═══════════════════════════════════════════════════
    
    private var isConnected = false
    private var connectedDevice: BluetoothDevice? = null
    
    private var bluetoothSocket: BluetoothSocket? = null
    private var inputStream: InputStream? = null
    private var outputStream: OutputStream? = null
    private var readThread: Thread? = null
    
    fun isConnected(): Boolean = isConnected
    fun getConnectedDevice(): BluetoothDevice? = if (isConnected) connectedDevice else null
    
    // ═══════════════════════════════════════════════════
    // 3. 连接模块
    // ═══════════════════════════════════════════════════
    
    /**
     * 连接到蓝牙设备
     * 注意：在后台线程执行，避免 ANR
     */
    fun connectDevice(device: BluetoothDevice) {
        Log.d(TAG, "→ 连接设备: ${device.name} (${device.address})")
        
        if (isConnected) disconnect()
        
        Thread {
            try {
                bluetoothAdapter?.cancelDiscovery()
                
                bluetoothSocket = device.createRfcommSocketToServiceRecord(
                    UUID.fromString(UUID_SERIAL_PORT)
                )
                bluetoothSocket?.connect()
                
                inputStream = bluetoothSocket?.inputStream
                outputStream = bluetoothSocket?.outputStream
                
                isConnected = true
                connectedDevice = device
                
                Log.d(TAG, "✓ 连接成功")
                sendBroadcast(ACTION_CONNECTION_STATE_CHANGED, true, null)
                startReadThread()
                
            } catch (e: IOException) {
                Log.e(TAG, "✗ 连接异常: ${e.message}")
                isConnected = false
                cleanup()
                sendBroadcast(ACTION_CONNECTION_FAILED, false, e.message)
            }
        }.start()
    }
    
    /**
     * 断开连接
     */
    fun disconnect() {
        if (!isConnected) return
        
        Log.d(TAG, "→ 断开连接")
        
        try {
            isConnected = false
            readThread?.interrupt()
            inputStream?.close()
            outputStream?.close()
            bluetoothSocket?.close()
            cleanup()
            sendBroadcast(ACTION_CONNECTION_STATE_CHANGED, false, null)
            Log.d(TAG, "✓ 已断开")
        } catch (e: IOException) {
            Log.e(TAG, "✗ 断开异常: ${e.message}")
        }
    }
    
    private fun cleanup() {
        try {
            inputStream?.close()
            outputStream?.close()
            bluetoothSocket?.close()
        } catch (e: IOException) {}
        inputStream = null
        outputStream = null
        bluetoothSocket = null
        connectedDevice = null
    }
    
    // ═══════════════════════════════════════════════════
    // 4. 数据通信模块
    // ═══════════════════════════════════════════════════
    
    /**
     * 发送字节数据
     */
    fun sendData(data: ByteArray): Boolean {
        if (!isConnected || outputStream == null) {
            Log.w(TAG, "✗ 未连接")
            return false
        }
        
        return try {
            outputStream?.write(data)
            outputStream?.flush()
            Log.d(TAG, "→ 发送: ${String(data)}")
            true
        } catch (e: IOException) {
            Log.e(TAG, "✗ 发送异常: ${e.message}")
            isConnected = false
            false
        }
    }
    
    /**
     * 发送文本
     */
    fun sendText(text: String): Boolean {
        return sendData(text.toByteArray())
    }
    
    /**
     * 启动读取线程
     * 持续监听来自蓝牙设备的数据
     */
    private fun startReadThread() {
        readThread = Thread {
            Log.d(TAG, "→ 读取线程启动")
            val buffer = ByteArray(1024)
            var bytes: Int
            
            while (isConnected) {
                try {
                    bytes = inputStream?.read(buffer) ?: 0
                    if (bytes > 0) {
                        val data = buffer.sliceArray(0 until bytes)
                        val text = String(data)
                        Log.d(TAG, "← 收到: $text")
                        sendBroadcast(ACTION_DATA_RECEIVED, true, text)
                    }
                } catch (e: IOException) {
                    if (isConnected) {
                        Log.e(TAG, "✗ 读取异常: ${e.message}")
                        isConnected = false
                        cleanup()
                        sendBroadcast(ACTION_CONNECTION_FAILED, false, e.message)
                    }
                    break
                }
            }
            Log.d(TAG, "✓ 读取线程退出")
        }.apply { start() }
    }
    
    // ═══════════════════════════════════════════════════
    // 5. 广播模块
    // ═══════════════════════════════════════════════════
    
    /**
     * 发送广播通知 Activity
     */
    private fun sendBroadcast(action: String, state: Boolean, data: String?) {
        val intent = Intent(action).apply {
            putExtra(EXTRA_STATE, state)
            if (data != null) {
                putExtra(EXTRA_DATA, data)
            }
        }
        LocalBroadcastManager.getInstance(this).sendBroadcast(intent)
        Log.d(TAG, "📢 发送广播: $action")
    }
}
```

### 3.2 BluetoothActivity (Activity)

```kotlin
package com.example.bluetooth

import android.Manifest
import android.bluetooth.BluetoothAdapter
import android.bluetooth.BluetoothManager
import android.content.BroadcastReceiver
import android.content.ComponentName
import android.content.Context
import android.content.Intent
import android.content.IntentFilter
import android.content.ServiceConnection
import android.os.Build
import android.os.Bundle
import android.os.IBinder
import android.util.Log
import androidx.activity.result.contract.ActivityResultContracts
import androidx.appcompat.app.AppCompatActivity
import androidx.core.content.ContextCompat
import androidx.localbroadcastmanager.content.LocalBroadcastManager
import androidx.lifecycle.lifecycleScope
import kotlinx.coroutines.launch

/**
 * BluetoothActivity - 蓝牙连接的主 Activity
 * 
 * 职责：
 * - 请求权限
 * - 绑定 Service
 * - 监听状态广播
 * - 更新 UI
 */
class BluetoothActivity : AppCompatActivity() {

    // ═══════════════════════════════════════════════════
    // 1. 基本属性
    // ═══════════════════════════════════════════════════
    
    private var bluetoothService: BluetoothService? = null
    private var isBound = false
    
    private val bluetoothAdapter by lazy {
        val manager = getSystemService(Context.BLUETOOTH_SERVICE) as BluetoothManager
        manager.adapter
    }
    
    companion object {
        private const val TAG = "BluetoothActivity"
    }
    
    // ═══════════════════════════════════════════════════
    // 2. 生命周期
    // ═══════════════════════════════════════════════════
    
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_bluetooth)
        
        Log.d(TAG, "→ Activity 创建")
        bindBluetoothService()
        registerBroadcastReceiver()
        checkAndRequestPermissions()
    }
    
    override fun onDestroy() {
        super.onDestroy()
        Log.d(TAG, "→ Activity 销毁")
        
        if (isBound) {
            unbindService(serviceConnection)
            isBound = false
        }
        LocalBroadcastManager.getInstance(this)
            .unregisterReceiver(broadcastReceiver)
    }
    
    // ═══════════════════════════════════════════════════
    // 3. 权限模块
    // ═══════════════════════════════════════════════════
    
    private val permissionLauncher = registerForActivityResult(
        ActivityResultContracts.RequestMultiplePermissions()
    ) { permissions ->
        val allGranted = permissions.values.all { it }
        if (allGranted) {
            Log.d(TAG, "✓ 权限已授予")
            startBluetoothDiscovery()
        } else {
            Log.w(TAG, "✗ 权限被拒绝")
        }
    }
    
    private fun checkAndRequestPermissions() {
        Log.d(TAG, "→ 检查权限")
        
        val permissions = mutableListOf<String>().apply {
            add(Manifest.permission.BLUETOOTH)
            add(Manifest.permission.BLUETOOTH_ADMIN)
            add(Manifest.permission.ACCESS_FINE_LOCATION)
            
            if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.S) {
                add(Manifest.permission.BLUETOOTH_SCAN)
                add(Manifest.permission.BLUETOOTH_CONNECT)
            }
        }
        
        val needRequest = permissions.any {
            ContextCompat.checkSelfPermission(this, it) !=
                    android.content.pm.PackageManager.PERMISSION_GRANTED
        }
        
        if (needRequest) {
            permissionLauncher.launch(permissions.toTypedArray())
        } else {
            startBluetoothDiscovery()
        }
    }
    
    private fun startBluetoothDiscovery() {
        Log.d(TAG, "→ 启动设备扫描")
        
        if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.S) {
            if (ContextCompat.checkSelfPermission(
                    this, Manifest.permission.BLUETOOTH_SCAN
                ) != android.content.pm.PackageManager.PERMISSION_GRANTED
            ) return
        }
        
        bluetoothAdapter?.startDiscovery()
    }
    
    // ═══════════════════════════════════════════════════
    // 4. Service 绑定模块
    // ═══════════════════════════════════════════════════
    
    private val serviceConnection = object : ServiceConnection {
        override fun onServiceConnected(name: ComponentName?, service: IBinder?) {
            Log.d(TAG, "✓ Service 已连接")
            bluetoothService = (service as BluetoothService.LocalBinder).getService()
            isBound = true
            updateConnectionUI()
        }
        
        override fun onServiceDisconnected(name: ComponentName?) {
            Log.d(TAG, "✗ Service 已断开")
            isBound = false
            bluetoothService = null
        }
    }
    
    private fun bindBluetoothService() {
        Log.d(TAG, "→ 绑定 Service")
        val intent = Intent(this, BluetoothService::class.java)
        startService(intent)  // 启动 Service
        bindService(intent, serviceConnection, Context.BIND_AUTO_CREATE)  // 绑定 Service
    }
    
    // ═══════════════════════════════════════════════════
    // 5. 广播模块
    // ═══════════════════════════════════════════════════
    
    private val broadcastReceiver = object : BroadcastReceiver() {
        override fun onReceive(context: Context?, intent: Intent?) {
            when (intent?.action) {
                BluetoothService.ACTION_CONNECTION_STATE_CHANGED -> {
                    val isConnected = intent.getBooleanExtra(
                        BluetoothService.EXTRA_STATE, false
                    )
                    onConnectionStateChanged(isConnected)
                }
                BluetoothService.ACTION_DATA_RECEIVED -> {
                    val data = intent.getStringExtra(BluetoothService.EXTRA_DATA) ?: ""
                    onDataReceived(data)
                }
                BluetoothService.ACTION_CONNECTION_FAILED -> {
                    val error = intent.getStringExtra(BluetoothService.EXTRA_ERROR)
                    onConnectionFailed(error ?: "未知错误")
                }
            }
        }
    }
    
    private fun registerBroadcastReceiver() {
        Log.d(TAG, "→ 注册广播接收器")
        val filter = IntentFilter().apply {
            addAction(BluetoothService.ACTION_CONNECTION_STATE_CHANGED)
            addAction(BluetoothService.ACTION_DATA_RECEIVED)
            addAction(BluetoothService.ACTION_CONNECTION_FAILED)
        }
        LocalBroadcastManager.getInstance(this)
            .registerReceiver(broadcastReceiver, filter)
    }
    
    // ═══════════════════════════════════════════════════
    // 6. 用户交互模块
    // ═══════════════════════════════════════════════════
    
    fun connectToDevice(deviceAddress: String) {
        if (!isBound) return
        
        Log.d(TAG, "→ 连接设备: $deviceAddress")
        val device = bluetoothAdapter?.getRemoteDevice(deviceAddress)
        device?.let { bluetoothService?.connectDevice(it) }
    }
    
    fun sendData(text: String) {
        if (!isBound) return
        
        Log.d(TAG, "→ 发送数据: $text")
        bluetoothService?.sendText(text)
    }
    
    fun disconnectDevice() {
        if (!isBound) return
        
        Log.d(TAG, "→ 断开连接")
        bluetoothService?.disconnect()
    }
    
    // ═══════════════════════════════════════════════════
    // 7. 回调处理
    // ═══════════════════════════════════════════════════
    
    private fun onConnectionStateChanged(isConnected: Boolean) {
        Log.d(TAG, "📊 连接状态: $isConnected")
        updateConnectionUI()
        
        if (isConnected) {
            bluetoothService?.getConnectedDevice()?.let {
                Log.d(TAG, "✓ 已连接到: ${it.name}")
            }
        }
    }
    
    private fun onDataReceived(data: String) {
        Log.d(TAG, "📨 收到数据: $data")
        lifecycleScope.launch {
            updateDataUI(data)
        }
    }
    
    private fun onConnectionFailed(error: String) {
        Log.e(TAG, "❌ 连接失败: $error")
        lifecycleScope.launch {
            showError(error)
        }
    }
    
    // ═══════════════════════════════════════════════════
    // 8. UI 更新（占位符）
    // ═══════════════════════════════════════════════════
    
    private fun updateConnectionUI() {

        // TODO: 更新 UI 控件
    }

        // TODO: 更新接收数据的 UI 控件
    }
    
    private fun showError(error: String) {

        // TODO: 显示错误提示
    }
}
```

```kotlin
private var dataBuffer = ""

private fun startReadThread() {
    readThread = Thread {
        val buffer = ByteArray(1024)
        var bytes: Int
        
        while (isConnected) {
            try {
                bytes = inputStream?.read(buffer) ?: 0
                
                if (bytes > 0) {
                    val text = String(buffer.sliceArray(0 until bytes))
                    dataBuffer += text
                    
                    // 按换行符分隔处理完整行
                    val lines = dataBuffer.split("\n")
                    for (i in 0 until lines.size - 1) {
                        val line = lines[i].trim()
                        if (line.isNotEmpty()) {
                            sendBroadcast(ACTION_DATA_RECEIVED, true, line)
                        }
                    }
                    
                    // 保留最后的不完整行
                    dataBuffer = lines.last()
                }
            } catch (e: IOException) {
                if (isConnected) {
                    isConnected = false
                    cleanup()
                    sendBroadcast(ACTION_CONNECTION_FAILED, false, e.message)
                }
                break
            }
        }
    }.apply { start() }
}
```

## 五、错误处理

### 错误 1: "connect failed"

**原因**

- 设备未配对
- 权限不足
- Socket 重复创建

**解决**

```kotlin
private fun cleanOldSocket() {
    try {
        bluetoothSocket?.close()
    } catch (e: Exception) {}
    bluetoothSocket = null
}

fun connectDevice(device: BluetoothDevice) {
    cleanOldSocket()  // 先清理旧 Socket
    // 然后连接...
}
```

### 错误 2: ANR (应用无响应)

**原因**

- 在主线程执行 connect()、read() 等阻塞操作

**解决**

```kotlin
// ❌ 错误
bluetoothSocket?.connect()  // 主线程 → ANR

// ✓ 正确
Thread {
    bluetoothSocket?.connect()  // 后台线程
}.start()
```

### 错误 3: 权限检查失败 (Android 12+)

**检查清单**

```kotlin
// ❌ 错误：只检查 Build.VERSION，不检查权限
if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.S) {
    // 这不是权限检查！
}

// ✓ 正确：使用 ContextCompat
if (ContextCompat.checkSelfPermission(
    this, 
    Manifest.permission.BLUETOOTH_CONNECT
) != PackageManager.PERMISSION_GRANTED) {
    // 权限未授予
}
```

### 错误 4: 数据不完整

**原因**

- InputStream.read() 一次可能读不完整
- 需要自己定义协议

**解决**

```kotlin
// 使用长度前缀协议
fun sendWithHeader(data: String): Boolean {
    val header = data.length.toString().padStart(4, '0')
    return sendText(header + data)
}
```

### 错误 5: Service 被系统杀死

**原因**

- 只用 bindService，未用 startService
- 系统内存不足

**解决**

```kotlin
// 同时启动和绑定
startService(intent)                    // 独立生命周期
bindService(intent, ..., BIND_AUTO_CREATE)  // 通信通道

// 或使用前台 Service
override fun onCreate() {
    super.onCreate()
    startForeground(NOTIFICATION_ID, createNotification())
}
```

### 错误 6: 内存泄漏

**检查清单**

```kotlin
override fun onDestroy() {
    super.onDestroy()
    
    // ✓ 必须 unbind Service
    if (isBound) {
        unbindService(serviceConnection)
    }
    
    // ✓ 必须 unregister Receiver
    LocalBroadcastManager.getInstance(this)
        .unregisterReceiver(broadcastReceiver)
    
    // ✓ Service 中必须关闭所有资源
    isConnected = false
    readThread?.interrupt()
    inputStream?.close()
    outputStream?.close()
    bluetoothSocket?.close()
}
```
