## 基础文本控件

### Text

**用途**：显示静态文本

**特点**：

- 最基础的文本显示控件
- 支持样式、颜色、字体大小等属性
- 支持行数限制、文本溢出处理

**使用示例**：

```kotlin
Text(
    text = "Hello, Compose!",
    fontSize = 16.sp,
    color = Color.Black,
    fontWeight = FontWeight.Bold,
    maxLines = 2,
    overflow = TextOverflow.Ellipsis
)
```

---

### SelectionContainer

**用途**：使文本可被选择和复制

**特点**：

- 包装Text使其可选择
- 通常用于需要用户复制内容的场景

**使用示例**：

```kotlin
SelectionContainer {
    Text("这是可以被选择的文本")
}
```

## 输入控件

### TextField

**用途**：单行文本输入框

**特点**：

- 支持输入验证
- 支持前缀/后缀图标
- 支持错误状态显示

**使用示例**：

```kotlin
var text by remember { mutableStateOf("") }

TextField(
    value = text,
    onValueChange = { text = it },
    label = { Text("输入用户名") },
    leadingIcon = { Icon(Icons.Default.Person, null) },
    isError = text.isEmpty(),
    modifier = Modifier.fillMaxWidth()
)
```

---

### OutlinedTextField

**用途**：带外框线的文本输入框（推荐用于 Material Design 3）

**特点**：

- Material Design 风格
- 比 TextField 视觉效果更轻
- 适合现代应用设计

**使用示例**：

```kotlin
var email by remember { mutableStateOf("") }

OutlinedTextField(
    value = email,
    onValueChange = { email = it },
    label = { Text("邮箱地址") },
    keyboardOptions = KeyboardOptions(keyboardType = KeyboardType.Email),
    modifier = Modifier.fillMaxWidth()
)
```

---

### BasicTextField

**用途**：无样式的文本输入框（低级 API）

**特点**：

- 完全自定义空间
- 需要手动实现样式
- 用于高度定制需求

**使用示例**：

```kotlin
var text by remember { mutableStateOf("") }

BasicTextField(
    value = text,
    onValueChange = { text = it },
    modifier = Modifier
        .fillMaxWidth()
        .border(1.dp, Color.Gray)
        .padding(8.dp)
)
```

## 按钮控件

### Button

**用途**：主要操作按钮

**特点**：

- Material Design 风格
- 默认背景色为主题颜色
- 包含点击反馈

**使用示例**：

```kotlin
Button(
    onClick = { /* 处理点击事件 */ },
    modifier = Modifier.fillMaxWidth()
) {
    Text("提交")
}
```

---

### OutlinedButton

**用途**：次要操作按钮

**特点**：

- 带外框线，无填充背景
- 用于次要操作
- 视觉优先级低于 Button

**使用示例**：

```kotlin
OutlinedButton(
    onClick = { /* 处理点击 */ }
) {
    Text("取消")
}
```

---

### TextButton

**用途**：文字按钮（优先级最低）

**特点**：

- 只显示文字，无背景
- 用于最低优先级操作
- 常用于对话框操作

**使用示例**：

```kotlin
TextButton(onClick = { /* 处理点击 */ }) {
    Text("了解更多")
}
```

---

### IconButton

**用途**：图标按钮

**特点**：

- 用于显示图标操作
- 通常用于工具栏、AppBar
- 需要结合 Icon 使用

**使用示例**：

```kotlin
IconButton(onClick = { /* 处理点击 */ }) {
    Icon(Icons.Default.Favorite, contentDescription = "收藏")
}
```

---

### FloatingActionButton (FAB)

**用途**：悬浮动作按钮

**特点**：

- 圆形浮动按钮
- 通常用于主要操作
- 支持多个子操作（ExtendedFloatingActionButton）

**使用示例**：

```kotlin
FloatingActionButton(
    onClick = { /* 新建操作 */ },
    modifier = Modifier.align(Alignment.BottomEnd)
) {
    Icon(Icons.Default.Add, "新建")
}
```

---

### ExtendedFloatingActionButton

**用途**：扩展的浮动按钮（带文字）

**使用示例**：

```kotlin
ExtendedFloatingActionButton(
    onClick = { /* 处理点击 */ },
    icon = { Icon(Icons.Default.Add, null) },
    text = { Text("新建") }
)
```

## 容器布局控件

### Column

**用途**：垂直线性布局

**特点**：

- 子元素垂直排列
- 支持对齐和间距设置
- 最常用的布局控件

**使用示例**：

```kotlin
Column(
    modifier = Modifier
        .fillMaxWidth()
        .padding(16.dp),
    verticalArrangement = Arrangement.spacedBy(8.dp),
    horizontalAlignment = Alignment.CenterHorizontally
) {
    Text("标题")
    Text("内容")
    Button(onClick = {}) { Text("按钮") }
}
```

---

### Row

**用途**：水平线性布局

**特点**：

- 子元素水平排列
- 支持权重分配
- 常用于并排放置控件

**使用示例**：

```kotlin
Row(
    modifier = Modifier.fillMaxWidth(),
    horizontalArrangement = Arrangement.SpaceBetween,
    verticalAlignment = Alignment.CenterVertically
) {
    Text("名称")
    Text("值")
}
```

---

### Box

**用途**：堆叠布局（最灵活）

**特点**：

- 可层叠放置子元素
- 支持 Z-order 控制
- 用于复杂的重叠布局

**使用示例**：

```kotlin
Box(
    modifier = Modifier
        .size(200.dp)
        .background(Color.Blue)
) {
    Text("背景", Modifier.align(Alignment.Center))
    Icon(
        Icons.Default.Close,
        null,
        modifier = Modifier.align(Alignment.TopEnd)
    )
}
```

---

### Surface

**用途**：提供表面和背景

**特点**：

- Material Design 表面
- 支持阴影和边框
- 用于创建卡片、面板等

**使用示例**：

```kotlin
Surface(
    shape = RoundedCornerShape(8.dp),
    shadowElevation = 4.dp,
    modifier = Modifier.padding(16.dp)
) {
    Text("内容", Modifier.padding(16.dp))
}
```

## 列表控件

### LazyColumn

**用途**：垂直滚动列表（性能优化）

**特点**：

- 只渲染可见项，性能高
- 支持大数据列表
- 支持粘性头部

**使用示例**：

```kotlin
LazyColumn(
    modifier = Modifier.fillMaxWidth(),
    verticalArrangement = Arrangement.spacedBy(8.dp)
) {
    items(itemList.size) { index ->
        Text("项目 ${index}")
    }
}
```

---

### LazyRow

**用途**：水平滚动列表

**特点**：

- 水平方向虚拟化滚动
- 用于水平列表（如图片轮播）

**使用示例**：

```kotlin
LazyRow(
    horizontalArrangement = Arrangement.spacedBy(8.dp)
) {
    items(categories) { category ->
        CategoryChip(category)
    }
}
```

---

### LazyVerticalGrid

**用途**：网格列表

**特点**：

- 二维网格布局
- 虚拟化渲染
- 用于图片网格等

**使用示例**：

```kotlin
LazyVerticalGrid(
    columns = GridCells.Fixed(3),
    verticalArrangement = Arrangement.spacedBy(8.dp),
    horizontalArrangement = Arrangement.spacedBy(8.dp)
) {
    items(imageList) { image ->
        AsyncImage(model = image, contentDescription = null)
    }
}
```

## 选择控件

### Checkbox

**用途**：复选框

**特点**：

- 二态选择（选中/未选中）
- 支持多选
- 支持不确定状态

**使用示例**：

```kotlin
var isChecked by remember { mutableStateOf(false) }

Checkbox(
    checked = isChecked,
    onCheckedChange = { isChecked = it }
)
```

---

### RadioButton

**用途**：单选按钮

**特点**：

- 单选模式
- 同一组中只能选一个
- 需配合按钮组使用

**使用示例**：

```kotlin
var selectedOption by remember { mutableStateOf("Option1") }
val options = listOf("Option1", "Option2", "Option3")

Column {
    options.forEach { option ->
        Row(verticalAlignment = Alignment.CenterVertically) {
            RadioButton(
                selected = selectedOption == option,
                onClick = { selectedOption = option }
            )
            Text(option)
        }
    }
}
```

---

### Switch

**用途**：开关控件

**特点**：

- 切换开/关状态
- 常用于设置页面
- 支持自定义样式

**使用示例**：

```kotlin
var isEnabled by remember { mutableStateOf(false) }

Switch(
    checked = isEnabled,
    onCheckedChange = { isEnabled = it }
)
```

---

### Slider

**用途**：滑块控制器

**特点**：

- 连续数值选择
- 支持范围显示
- 常用于音量、亮度调节

**使用示例**：

```kotlin
var volume by remember { mutableStateOf(0.5f) }

Slider(
    value = volume,
    onValueChange = { volume = it },
    valueRange = 0f..1f,
    modifier = Modifier.fillMaxWidth()
)
```

---

### RangeSlider

**用途**：范围滑块

**特点**：

- 同时选择最小值和最大值
- 用于价格范围等选择

**使用示例**：

```kotlin
var range by remember { mutableStateOf(0f..100f) }

RangeSlider(
    value = range,
    onValueChange = { range = it },
    valueRange = 0f..100f
)
```

---

### DropdownMenu

**用途**：下拉菜单

**特点**：

- 点击显示菜单选项
- 支持多个菜单项
- 常用于快速选择

**使用示例**：

```kotlin
var isExpanded by remember { mutableStateOf(false) }
var selectedItem by remember { mutableStateOf("选项1") }

Box {
    Button(onClick = { isExpanded = !isExpanded }) {
        Text(selectedItem)
    }
    DropdownMenu(
        expanded = isExpanded,
        onDismissRequest = { isExpanded = false }
    ) {
        DropdownMenuItem(
            text = { Text("选项1") },
            onClick = { selectedItem = "选项1"; isExpanded = false }
        )
        DropdownMenuItem(
            text = { Text("选项2") },
            onClick = { selectedItem = "选项2"; isExpanded = false }
        )
    }
}
```

## Dialog 和 Popup

### AlertDialog

**用途**：警告对话框

**特点**：

- 模态对话框
- 需要用户确认
- 常用于警告和确认

**使用示例**：

```kotlin
var showDialog by remember { mutableStateOf(false) }

if (showDialog) {
    AlertDialog(
        title = { Text("确认删除") },
        text = { Text("此操作无法撤销") },
        confirmButton = {
            Button(onClick = { 
                // 执行删除
                showDialog = false 
            }) {
                Text("删除")
            }
        },
        dismissButton = {
            TextButton(onClick = { showDialog = false }) {
                Text("取消")
            }
        },
        onDismissRequest = { showDialog = false }
    )
}

Button(onClick = { showDialog = true }) {
    Text("删除")
}
```

---

### BasicAlertDialog

**用途**：无样式对话框（低级 API）

**特点**：

- 完全自定义
- 需要手动实现样式

---

### Popup

**用途**：弹出窗口

**特点**：

- 在指定位置显示内容
- 点击外部自动关闭
- 用于浮层内容

**使用示例**：

```kotlin
var showPopup by remember { mutableStateOf(false) }

Box {
    Button(onClick = { showPopup = !showPopup }) {
        Text("显示弹窗")
    }
    
    if (showPopup) {
        Popup(
            alignment = Alignment.TopCenter,
            onDismissRequest = { showPopup = false }
        ) {
            Surface(
                shape = RoundedCornerShape(8.dp),
                shadowElevation = 8.dp
            ) {
                Text("弹窗内容", Modifier.padding(16.dp))
            }
        }
    }
}
```

```kotlin
var count by remember { mutableStateOf(0) }
```

```kotlin
var text by rememberSaveable { mutableStateOf("") }
```

```kotlin
@HiltViewModel
class MyViewModel : ViewModel() {
    private val _uiState = MutableStateFlow(MyUiState())
    val uiState: StateFlow<MyUiState> = _uiState.asStateFlow()
}

@Composable
fun MyScreen(viewModel: MyViewModel) {
    val uiState by viewModel.uiState.collectAsState()
    // 使用 uiState
}
```

## 常用修饰符

|修饰符|用途|示例|

## 关键要点

1. **性能优化**：使用 `LazyColumn`、`LazyRow` 代替 `Column`、`Row` 处理大列表
2. **状态管理**：优先使用 `remember` 和 `rememberSaveable`
3. **可访问性**：为所有有交互的控件提供 `contentDescription`
