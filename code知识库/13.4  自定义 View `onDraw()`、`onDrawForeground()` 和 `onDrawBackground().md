**自定义 View 示例：重写 `onDraw()`、`onDrawForeground()` 和 `onDrawBackground()`**

以下是一个完整的自定义 View 示例，演示如何在 `onDrawBackground`、`onDraw` 和 `onDrawForeground` 中进行绘制。

```java
package com.example.customview;

import android.content.Context;
import android.graphics.Canvas;
import android.graphics.Color;
import android.graphics.Paint;
import android.graphics.RectF;
import android.os.Build;
import android.util.AttributeSet;
import android.view.View;

import androidx.annotation.Nullable;
import androidx.annotation.RequiresApi;

public class CustomDrawView extends View {

    private Paint paint;

    public CustomDrawView(Context context) {
        this(context, null);
    }

    public CustomDrawView(Context context, @Nullable AttributeSet attrs) {
        this(context, attrs, 0);
    }

    public CustomDrawView(Context context, @Nullable AttributeSet attrs, int defStyleAttr) {
        super(context, attrs, defStyleAttr);
        init();
    }

    private void init() {
        paint = new Paint(Paint.ANTI_ALIAS_FLAG);
        paint.setStyle(Paint.Style.FILL);
        // 默认背景色可通过 setBackgroundColor 设置，这里仅演示手动绘制
    }

    /**
     * 绘制背景（通常用于绘制自定义背景，优先级最低）
     * 注意：系统默认会先调用 super.onDrawBackground(canvas) 来绘制 setBackground 设定的背景
     */
    @Override
    protected void onDrawBackground(Canvas canvas) {
        super.onDrawBackground(canvas); // 保留系统背景绘制

        // 示例：在背景层画一个浅色矩形
        paint.setColor(Color.parseColor("#E0F7FA"));
        canvas.drawRect(0, 0, getWidth(), getHeight(), paint);

        // 示例：画背景装饰（网格或图案）
        paint.setColor(Color.parseColor("#B2EBF2"));
        paint.setStrokeWidth(2);
        for (int i = 0; i < getWidth(); i += 40) {
            canvas.drawLine(i, 0, i, getHeight(), paint);
        }
        for (int i = 0; i < getHeight(); i += 40) {
            canvas.drawLine(0, i, getWidth(), i, paint);
        }
    }

    /**
     * 主绘制方法（最常用）
     */
    @Override
    protected void onDraw(Canvas canvas) {
        super.onDraw(canvas);

        // 示例1：绘制一个带边框的圆角矩形
        paint.setColor(Color.parseColor("#FF4081"));
        RectF rect = new RectF(100, 100, getWidth() - 100, getHeight() - 200);
        canvas.drawRoundRect(rect, 40, 40, paint);

        // 示例2：绘制文字
        paint.setColor(Color.WHITE);
        paint.setTextSize(60);
        paint.setTextAlign(Paint.Align.CENTER);
        canvas.drawText("自定义View", getWidth() / 2f, getHeight() / 2f, paint);

        // 示例3：绘制圆形
        paint.setColor(Color.parseColor("#3F51B5"));
        canvas.drawCircle(getWidth() / 2f, 300, 80, paint);
    }

    /**
     * 绘制前景（API 23+），通常用于覆盖在内容之上的装饰（如阴影、边框、徽章等）
     * 优先级最高，会绘制在 onDraw 之上
     */
    @RequiresApi(api = Build.VERSION_CODES.M)
    @Override
    public void onDrawForeground(Canvas canvas) {
        super.onDrawForeground(canvas);

        // 示例：绘制半透明前景遮罩
        paint.setColor(Color.parseColor("#80000000")); // 半透明黑色
        canvas.drawRect(0, 0, getWidth(), getHeight(), paint);

        // 示例：绘制高亮边框
        paint.setColor(Color.YELLOW);
        paint.setStyle(Paint.Style.STROKE);
        paint.setStrokeWidth(12);
        canvas.drawRoundRect(40, 40, getWidth() - 40, getHeight() - 40, 30, 30, paint);

        // 示例：绘制右上角的角标
        paint.setStyle(Paint.Style.FILL);
        paint.setColor(Color.RED);
        canvas.drawCircle(getWidth() - 80, 80, 40, paint);

        paint.setColor(Color.WHITE);
        paint.setTextSize(36);
        paint.setTextAlign(Paint.Align.CENTER);
        canvas.drawText("NEW", getWidth() - 80, 88, paint);
    }

    @Override
    protected void onMeasure(int widthMeasureSpec, int heightMeasureSpec) {
        // 简单示例：给一个默认尺寸
        int defaultWidth = 600;
        int defaultHeight = 400;

        int width = MeasureSpec.getSize(widthMeasureSpec);
        int height = MeasureSpec.getSize(heightMeasureSpec);

        setMeasuredDimension(
                resolveSize(defaultWidth, widthMeasureSpec),
                resolveSize(defaultHeight, heightMeasureSpec)
        );
    }
}
```

### 使用方法

在 XML 布局中：

```xml
<com.example.customview.CustomDrawView
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:background="#F0F0F0" />   <!-- 系统背景会被 onDrawBackground 看到 -->
```

### 绘制顺序（重要）

1. `onDrawBackground(Canvas)` —— 背景层
2. `onDraw(Canvas)` —— 主体内容
3. `onDrawForeground(Canvas)` —— 前景层（最上层）

**注意事项**：
- `onDrawBackground` 很少被直接重写，多数情况使用 `setBackground()` 或 `setBackgroundDrawable()`。
- `onDrawForeground` 需要 API 23（Android 6.0）以上。
- 所有绘制都应尽量使用已初始化的 `Paint` 对象，并注意性能（避免在 `onDraw` 中频繁创建对象）。
- 建议在 `onSizeChanged` 中缓存需要根据尺寸计算的矩形/路径等。
