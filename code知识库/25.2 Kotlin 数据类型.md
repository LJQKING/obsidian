Kotlin 是**强类型语言**，所有数据类型本质上都继承自 `Any`（类似 Java 的 `Object`）。

---

# Kotlin 数据类型分类

```text
Any
 ├── Number
 │    ├── Byte
 │    ├── Short
 │    ├── Int
 │    ├── Long
 │    ├── Float
 │    └── Double
 │
 ├── Char
 ├── Boolean
 ├── String
 │
 ├── Array
 ├── Collection
 │    ├── List
 │    ├── Set
 │    └── Map
 │
 └── Nullable Type
```

# 1. 整型（Integer）

|类型|大小|
|---|---|
|Byte|8位|
|Short|16位|
|Int|32位|
|Long|64位|

## Byte

```kotlin
val a: Byte = 100
```

---

## Short

```kotlin
val b: Short = 1000
```

---

## Int（最常用）

```kotlin
val age: Int = 18
```

类型推断：

```kotlin
val age = 18
```

自动推断为：

```kotlin
Int
```

---

## Long

必须加 `L`

```kotlin
val time = 10000000000L
```

---

# 2. 浮点型

## Float

必须加 `f`

```kotlin
val score = 98.5f
```

---

## Double（默认）

```kotlin
val price = 99.99
```

Kotlin 默认浮点数：

```kotlin
Double
```

# 3. 字符类型 Char

只能存一个字符。

```kotlin
val c: Char = 'A'
```

```kotlin
val c = '中'
```

```kotlin
val isVip = false
```

逻辑运算：

```kotlin
&&
||
!
```

```kotlin

}
```

# 5. String（最常用）

## 定义

```kotlin
val name = "Tom"
```

---

## 字符串模板

Java：

```java
String msg = "Name=" + name;
```

Kotlin：

```kotlin
val msg = "Name=$name"
```

---

复杂表达式：

```kotlin
val age = 18

println("Age=${age + 1}")
```

输出：

```text
Age=19
```

---

## 多行字符串

```kotlin
val json = """
{
    "name":"Tom",
    "age":18
}
"""
```

Android 解析 JSON 时很方便。

# 7. 集合类型

---

## List

有序可重复

```kotlin
val list = listOf(
    "Tom",
    "Jack",
    "Lucy"
)
```

遍历：

```kotlin
list.forEach {
    println(it)
}
```

---

## MutableList

可修改

```kotlin
val list = mutableListOf<String>()

list.add("Tom")
list.add("Jack")
```

---

## Set

不可重复

```kotlin
val set = setOf(
    "A",
    "B",
    "A"
)
```

结果：

```text
A
B
```

---

## Map

键值对

```kotlin
val map = mapOf(
    "name" to "Tom",
    "age" to 18
)
```

访问：

```kotlin
println(map["name"])
```

# 8. Any

类似 Java Object。

```kotlin
val data: Any = "Hello"
```

也可以：

```kotlin
val data: Any = 100
```

---

类型判断：

```kotlin
if (data is String) {
    println(data.length)
}
```

Kotlin 自动类型转换（Smart Cast）。

---

# 9. Unit

类似 Java 的 void。

```kotlin
fun test(): Unit {
}
```

简写：

```kotlin
fun test() {
}
```

---

# 10. Nothing

表示永远不会返回。

例如：

```kotlin
fun crash(): Nothing {
    throw RuntimeException()
}
```

或者：

```kotlin
fun exit(): Nothing {
    while(true){}
}
```

用途：

- 异常处理
    
- Flow
    
- 协程源码
    

---

# 11. Nullable（可空类型）

Kotlin 最大特点之一。

---

## 不可空

```kotlin
var name: String = "Tom"
```

错误：

```kotlin
name = null
```

---

## 可空

```kotlin
var name: String? = null
```

---

## 安全调用

```kotlin
name?.length
```

---

## Elvis操作符

```kotlin
val len = name?.length ?: 0
```

等价：

```kotlin
if(name != null)
    name.length
else
    0
```

# Android 面试高频考点

### 1. Int 和 Integer 区别

Kotlin：

```kotlin
val age = 18
```

编译后：

```java
int age = 18;
```

可空：

```kotlin
val age: Int? = 18
```

编译后：

```java
Integer age = 18;
```

---

### 2. String? 和 String 区别

```kotlin
String
```

不能为 null。

```kotlin
String?
```

允许为 null。

---

### 3. Any、Unit、Nothing 区别

|类型|作用|
|---|---|
|Any|所有类父类|
|Unit|无返回值|
|Nothing|永不返回|

---

### 4. Android 开发最常用类型

1. `String`
    
2. `Int`
    
3. `Boolean`
    
4. `List<T>`
    
5. `MutableList<T>`
    
6. `Map<K,V>`
    
7. `Array<T>`
    
8. `Any`
    
9. `String?`
    
10. `Unit`
    

掌握这些类型以及可空类型（`?`）、安全调用（`?.`）、Elvis 操作符（`?:`）和集合 API（`map`、`filter`、`flatMap` 等），基本覆盖了 Android Kotlin 开发中的大部分场景。
