**Android 自定义 View 绘制表格** 是常见的自定义控件需求。下面给你一个完整的、可直接使用的 **自定义 TableView** 示例，使用 `Canvas` 绘制网格和文本，支持行列数据、边框、单元格文本居中等基本功能。

### 1. 创建自定义 View 类（Kotlin 示例，推荐）

```kotlin
// TableView.kt
import android.content.Context
import android.graphics.Canvas
import android.graphics.Color
import android.graphics.Paint
import android.graphics.Rect
import android.util.AttributeSet
import android.view.View

class TableView @JvmOverloads constructor(
    context: Context,
    attrs: AttributeSet? = null,
    defStyleAttr: Int = 0
) : View(context, attrs, defStyleAttr) {

    private val paint = Paint(Paint.ANTI_ALIAS_FLAG).apply {
        style = Paint.Style.STROKE
        strokeWidth = 2f
        color = Color.BLACK
    }

    private val textPaint = Paint(Paint.ANTI_ALIAS_FLAG).apply {
        style = Paint.Style.FILL
        textSize = 40f
        color = Color.BLACK
        textAlign = Paint.Align.CENTER
    }

    // 数据：二维列表，每行是一个 List<String>
    var data: List<List<String>> = emptyList()
        set(value) {
            field = value
            requestLayout() // 重新测量
            invalidate()    // 重绘
        }

    private var cellWidth = 0f
    private var cellHeight = 0f
    private val textBounds = Rect()

    override fun onMeasure(widthMeasureSpec: Int, heightMeasureSpec: Int) {
        super.onMeasure(widthMeasureSpec, heightMeasureSpec)
        // 可根据数据动态计算建议尺寸，这里简单处理
        val suggestedWidth = if (data.isNotEmpty() && data[0].isNotEmpty()) {
            data[0].size * 200
        } else 800
        val suggestedHeight = data.size * 100
        setMeasuredDimension(
            resolveSize(suggestedWidth, widthMeasureSpec),
            resolveSize(suggestedHeight, heightMeasureSpec)
        )
    }

    override fun onSizeChanged(w: Int, h: Int, oldw: Int, oldh: Int) {
        super.onSizeChanged(w, h, oldw, oldh)
        if (data.isNotEmpty() && data[0].isNotEmpty()) {
            cellWidth = w.toFloat() / data[0].size
            cellHeight = h.toFloat() / data.size
        }
    }

    override fun onDraw(canvas: Canvas) {
        super.onDraw(canvas)
        if (data.isEmpty()) return

        val rows = data.size
        val cols = data[0].size

        // 绘制垂直线
        for (i in 0..cols) {
            val x = i * cellWidth
            canvas.drawLine(x, 0f, x, height.toFloat(), paint)
        }

        // 绘制水平线
        for (i in 0..rows) {
            val y = i * cellHeight
            canvas.drawLine(0f, y, width.toFloat(), y, paint)
        }

        // 绘制文本
        for (row in data.indices) {
            for (col in data[row].indices) {
                val text = data[row][col]
                textPaint.getTextBounds(text, 0, text.length, textBounds)

                val x = col * cellWidth + cellWidth / 2
                val y = row * cellHeight + cellHeight / 2 + textBounds.height() / 2

                canvas.drawText(text, x, y, textPaint)
            }
        }
    }

    // 可选：设置表头样式（不同 Paint）
    fun setHeaderTextColor(color: Int) {
        // 可扩展
    }
}
```

### 2. 在布局中使用（XML）

```xml
<com.yourpackage.TableView
    android:id="@+id/tableView"
    android:layout_width="match_parent"
    android:layout_height="wrap_content"
    android:background="#F5F5F5" />
```

### 3. Activity / Fragment 中设置数据

```kotlin
val tableView = findViewById<TableView>(R.id.tableView)

val tableData = listOf(
    listOf("姓名", "年龄", "城市"),      // 表头
    listOf("张三", "28", "北京"),
    listOf("李四", "35", "上海"),
    listOf("王五", "22", "广州")
)

tableView.data = tableData
```

### 进阶建议

1. **支持滚动**：继承 `HorizontalScrollView + VerticalScrollView` 嵌套，或直接继承 `RecyclerView`（更推荐复杂表格）。

2. **合并单元格**：记录特殊格子的 `rowSpan` / `colSpan`，绘制时跳过对应线条。

3. **点击事件**：重写 `onTouchEvent`，根据点击坐标 `(x/cellWidth, y/cellHeight)` 判断单元格。

4. **性能优化**：
   - 复杂表格推荐使用现有库（如 `TableView`）而不是纯 Canvas。
   - 大数据量时考虑 `Bitmap` 离屏缓存或 `RecyclerView` + `ItemDecoration`。

5. **自定义属性**（attrs.xml）：
   ```xml
   <declare-styleable name="TableView">
       <attr name="cellTextSize" format="dimension" />
       <attr name="borderColor" format="color" />
       <attr name="headerColor" format="color" />
   </declare-styleable>
   ```
