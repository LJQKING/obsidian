**在 Android 自定义 View 中，绘制完成后刷新数据的常见做法**

自定义 View 的核心绘制方法是 `onDraw(Canvas canvas)`，但 Android 并没有直接的“绘制完成”回调（因为绘制是系统调度，可能多次执行）。**刷新数据通常通过数据变更 → 调用 `invalidate()` / `postInvalidate()` → 重新触发 `onDraw()`** 来实现。

### 1. 最推荐的做法：数据驱动 + `invalidate()`

```kotlin
class MyCustomView @JvmOverloads constructor(
    context: Context, attrs: AttributeSet? = null
) : View(context, attrs) {

    private var dataList: List<SomeData> = emptyList()  // 你的数据

    // 更新数据的方法（推荐从外部调用）
    fun refreshData(newData: List<SomeData>) {
        this.dataList = newData
        // 数据变化后请求重绘
        invalidate()           // UI 线程调用
        // postInvalidate()    // 非 UI 线程调用
    }

    override fun onDraw(canvas: Canvas) {
        super.onDraw(canvas)
        // 使用最新 dataList 进行绘制
        dataList.forEach { item ->
            // 绘制逻辑...
        }
        
        // 【注意】不要在这里直接修改 dataList 并再次 invalidate()
        // onDraw 应该保持轻量，避免递归或卡顿
    }
}
```

**使用示例**：
```kotlin
myCustomView.refreshData(updatedList)
```

### 2. 如果确实需要在“绘制完成后”做事情

- **在 `onDraw` 末尾**（不推荐频繁使用）：
  ```kotlin
  override fun onDraw(canvas: Canvas) {
      super.onDraw(canvas)
      // ... 正常绘制
      
      // 绘制结束后做一次性操作（例如通知外部）
      drawingFinishedListener?.onDrawingComplete()
  }
  ```

- **更好的方式**：使用 `ViewTreeObserver` 或 `onSizeChanged` / `onAttachedToWindow` 监听布局/绘制时机。

```kotlin
view.viewTreeObserver.addOnGlobalLayoutListener {
    // 布局完成后（绘制即将或已经开始）
    refreshData(...)
}
```

或者使用 `post` 延迟到下一帧：
```kotlin
post {
    refreshData(newData)
}
```

### 3. 常见问题与优化

- **onDraw 不执行**：检查是否调用了 `invalidate()`；是否在非 UI 线程（用 `postInvalidate()`）；是否启用了硬件加速冲突；View 是否可见（`visibility = VISIBLE`）。
- **性能问题**：不要在 `onDraw` 里创建对象（Paint、Path 等应在构造函数或 `onSizeChanged` 中初始化）。使用局部刷新 `invalidate(Rect dirty)` 减少绘制区域。
- **需要重新测量/布局**：同时调用 `requestLayout()` + `invalidate()`。
- **异步数据**：用 `ViewModel + LiveData` 或 `Flow` 观察数据，数据变化时调用 `refreshData()`。

### 4. 完整小例子（Kotlin）

```kotlin
class RefreshableCustomView(context: Context, attrs: AttributeSet?) : View(context, attrs) {
    private var count = 0
    private val paint = Paint(Paint.ANTI_ALIAS_FLAG).apply {
        textSize = 50f
        color = Color.BLACK
    }

    fun incrementAndRefresh() {
        count++
        invalidate()
    }

    override fun onDraw(canvas: Canvas) {
        super.onDraw(canvas)
        canvas.drawText("计数: $count", 100f, 100f, paint)
        // 数据刷新逻辑写在这里或外部
    }
}
```
