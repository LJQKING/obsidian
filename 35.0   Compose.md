**Android Jetpack Compose 主要作用**：

**Jetpack Compose** 是 Google 推出的 **现代 Android 原生 UI 开发工具包**（Declarative UI Toolkit）。它取代了传统的 XML + View 系统，让开发者用 **纯 Kotlin 代码** 以声明式（Declarative）的方式构建界面。

### 核心特点（为什么用它）
- **声明式编程**：你描述“UI 应该是什么样子”，而不是一步步“怎么去改 UI”。
- **响应式**：状态改变时，UI 自动更新（基于 Composable 函数）。
- **更简洁、高性能**：代码量大幅减少，支持 Kotlin 的所有特性（协程、扩展函数等）。
- **完全兼容**：可以和传统 View 系统混合使用。
- **跨平台潜力**：Compose Multiplatform 可用于 Desktop、Web、iOS 等。

简单说：**它就是用来高效构建 Android App 界面的**，从简单按钮到复杂列表、动画、响应式布局都能轻松实现。

### Compose 内容分类（常用模块）

Compose 的 API 和组件可以按以下方式分类，便于学习和查阅：

#### 1. **基础构建块（Core Fundamentals）**
   - `Composable` 函数（核心）
   - `Modifier`（修改器：控制大小、padding、点击、背景等）
   - `State` 与 `MutableState`（状态管理：`remember`、`viewModel` 等）

#### 2. **布局（Layout）**
   - `Column`、`Row`、`Box`
   - `ConstraintLayout`、`Scaffold`（Material 脚手架）
   - `LazyColumn`、`LazyRow`（高效列表/网格）
   - `FlowRow`、`FlowColumn` 等

#### 3. **Material Design 组件（Material Components）**
官方按功能分类（Material 3）：
   - **Actions**：Button、FloatingActionButton、IconButton 等
   - **Communication**：Badge、Snackbar、Tooltip 等
   - **Containment**：Card、Surface、Dialog 等
   - **Navigation**：NavigationBar、TabRow、Drawer 等
   - **Selection**：Checkbox、RadioButton、Switch、Slider 等
   - **Text Inputs**：TextField、OutlinedTextField 等
   - **Progress**：ProgressIndicator、CircularProgressIndicator
   - **其他**：Chip、Menu、BottomSheet 等

#### 4. **动画与图形（Animation & Graphics）**
   - `AnimatedVisibility`、`animate*AsState`
   - `Transition`、`updateTransition`
   - `Canvas`、`draw*`（自定义绘制）
   - Lottie、Vector 等

#### 5. **状态管理与副作用（State & Side Effects）**
   - `remember`、`LaunchedEffect`、`DisposableEffect`
   - `ViewModel` + `StateFlow`/`LiveData`
   - `CompositionLocal`（上下文传递）

#### 6. **主题与样式（Theming）**
   - `MaterialTheme`、`ColorScheme`、`Typography`、`Shapes`
   - 动态颜色（Dynamic Color）、暗黑模式支持

#### 7. **高级与集成**
   - Navigation Compose（页面导航）
   - Accompanist（第三方扩展，如 Pager、Permissions）
   - Compose UI 测试、预览（@Preview）
   - 与 View 互操作

### **1. 入门：Text、Button、Column、Modifier + 简单状态**

```kotlin
@Composable
fun GreetingScreen() {
    var count by remember { mutableStateOf(0) }  // 简单状态

    Column(
        modifier = Modifier
            .fillMaxSize()
            .padding(16.dp),
        horizontalAlignment = Alignment.CenterHorizontally,
        verticalArrangement = Arrangement.Center
    ) {
        Text(
            text = "你点击了 $count 次",
            fontSize = 24.sp,
            fontWeight = FontWeight.Bold
        )

        Spacer(modifier = Modifier.height(16.dp))

        Button(onClick = { count++ }) {
            Text("点我 +1")
        }

        OutlinedButton(onClick = { count = 0 }) {
            Text("重置")
        }
    }
}
```

**关键点**：
- `remember { mutableStateOf() }` 管理状态
- `Column` + `Modifier` 布局
- `Button` 点击事件

### **2. 进阶：LazyColumn + ViewModel + 列表**

**ViewModel：**
```kotlin
class TodoViewModel : ViewModel() {
    private val _todos = mutableStateListOf<String>()
    val todos: List<String> = _todos

    fun addTodo(text: String) {
        if (text.isNotBlank()) _todos.add(text)
    }

    fun removeTodo(index: Int) {
        _todos.removeAt(index)
    }
}
```

**UI 部分：**
```kotlin
@Composable
fun TodoScreen(viewModel: TodoViewModel = viewModel()) {
    var input by remember { mutableStateOf("") }

    Column(modifier = Modifier.fillMaxSize()) {
        // 输入栏
        Row(
            modifier = Modifier.padding(16.dp),
            verticalAlignment = Alignment.CenterVertically
        ) {
            TextField(
                value = input,
                onValueChange = { input = it },
                modifier = Modifier.weight(1f),
                label = { Text("输入待办事项") }
            )
            Spacer(Modifier.width(8.dp))
            Button(onClick = {
                viewModel.addTodo(input)
                input = ""
            }) {
                Text("添加")
            }
        }

        // 列表
        LazyColumn {
            itemsIndexed(viewModel.todos) { index, todo ->
                Card(
                    modifier = Modifier
                        .fillMaxWidth()
                        .padding(horizontal = 16.dp, vertical = 4.dp)
                ) {
                    Row(
                        modifier = Modifier.padding(16.dp),
                        verticalAlignment = Alignment.CenterVertically
                    ) {
                        Text(text = todo, modifier = Modifier.weight(1f))
                        IconButton(onClick = { viewModel.removeTodo(index) }) {
                            Icon(Icons.Default.Delete, contentDescription = "删除")
                        }
                    }
                }
            }
        }
    }
}
```

**关键点**：
- `LazyColumn` + `itemsIndexed` 高效列表
- `ViewModel` + `mutableStateListOf`
- `Card` + `Row` 组合

### **3. 实用：Scaffold + Navigation + 表单（TextField）**

**主页面（带底部导航）：**
```kotlin
@Composable
fun MainApp() {
    val navController = rememberNavController()

    Scaffold(
        bottomBar = {
            NavigationBar {
                NavigationBarItem(
                    icon = { Icon(Icons.Default.Home, null) },
                    label = { Text("首页") },
                    selected = true,
                    onClick = { }
                )
                NavigationBarItem(
                    icon = { Icon(Icons.Default.List, null) },
                    label = { Text("列表") },
                    selected = false,
                    onClick = { }
                )
            }
        }
    ) { paddingValues ->
        NavHost(
            navController = navController,
            startDestination = "home",
            modifier = Modifier.padding(paddingValues)
        ) {
            composable("home") { HomeScreen() }
            composable("list") { TodoScreen() }
        }
    }
}
```

**表单示例（登录/提交表单）：**
```kotlin
@Composable
fun LoginForm() {
    var username by remember { mutableStateOf("") }
    var password by remember { mutableStateOf("") }
    var agreed by remember { mutableStateOf(false) }

    Column(
        modifier = Modifier
            .fillMaxSize()
            .padding(24.dp),
        verticalArrangement = Arrangement.spacedBy(16.dp)
    ) {
        TextField(
            value = username,
            onValueChange = { username = it },
            label = { Text("用户名") },
            modifier = Modifier.fillMaxWidth()
        )

        OutlinedTextField(
            value = password,
            onValueChange = { password = it },
            label = { Text("密码") },
            visualTransformation = PasswordVisualTransformation(),
            modifier = Modifier.fillMaxWidth()
        )

        Row(verticalAlignment = Alignment.CenterVertically) {
            Checkbox(checked = agreed, onCheckedChange = { agreed = it })
            Text("我已同意用户协议")
        }

        Button(
            onClick = { /* 登录逻辑 */ },
            modifier = Modifier.fillMaxWidth(),
            enabled = username.isNotBlank() && password.length >= 6 && agreed
        ) {
            Text("登录")
        }
    }
}
```

### **4. 高级：自定义 Layout、动画、Canvas**

**简单动画示例**：
```kotlin
@Composable
fun AnimatedBox() {
    var expanded by remember { mutableStateOf(false) }

    val size by animateDpAsState(if (expanded) 200.dp else 100.dp)

    Box(
        modifier = Modifier
            .size(size)
            .background(if (expanded) Color.Blue else Color.Red)
            .clickable { expanded = !expanded }
            .animateContentSize()  // 自动动画
    )
}
```

**Canvas 自定义绘制**（简单进度环）：
```kotlin
@Composable
fun CustomProgress(progress: Float) {
    Canvas(modifier = Modifier.size(120.dp)) {
        drawArc(
            color = Color.Gray,
            startAngle = 0f,
            sweepAngle = 360f,
            useCenter = false,
            style = Stroke(12f)
        )
        drawArc(
            color = Color.Green,
            startAngle = -90f,
            sweepAngle = progress * 360f,
            useCenter = false,
            style = Stroke(12f)
        )
    }
}
```
