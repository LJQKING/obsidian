**Kotlin 高级函数（高阶函数）详解**

Kotlin 中**高阶函数（Higher-Order Function）** 是函数式编程的核心特性之一。它允许**函数作为参数传递**、**函数作为返回值返回**，极大提升了代码的灵活性和抽象能力。

### 1. 什么是高阶函数？

如果一个函数：
- **接收** 其他函数作为参数
- **返回** 一个函数

那么它就是高阶函数。

### 2. 函数类型（Function Type）

Kotlin 用函数类型来描述函数的签名：

```kotlin
// (参数类型1, 参数类型2) -> 返回值类型
(Int, Int) -> Int
() -> Unit
(String) -> Boolean
```

### 3. Lambda 表达式

最常用的传递函数的方式：

```kotlin
val sum: (Int, Int) -> Int = { a, b -> a + b }

// 简化写法（类型推断）
val sum = { a: Int, b: Int -> a + b }
```

**常用语法糖**：

```kotlin
// 如果 lambda 是最后一个参数，可以写在括号外面
listOf(1,2,3,4).filter { it > 2 }

// it 关键字（单个参数时）
listOf(1,2,3).map { it * 2 }
```

### 4. 高阶函数示例

#### **标准库中的经典高阶函数**

```kotlin
// filter、map、forEach 等都是高阶函数
val numbers = listOf(1, 2, 3, 4, 5, 6)

val even = numbers.filter { it % 2 == 0 }
val squared = numbers.map { it * it }
numbers.forEach { println(it) }

// let、run、apply、also、with
val result = "hello".let {
    it.uppercase() + "!"
}
```

#### **自定义高阶函数**

```kotlin
// 接收函数作为参数
fun <T> List<T>.customFilter(predicate: (T) -> Boolean): List<T> {
    val result = mutableListOf<T>()
    forEach { if (predicate(it)) result.add(it) }
    return result
}

// 返回一个函数
fun multiplier(factor: Int): (Int) -> Int {
    return { it * factor }
}

val double = multiplier(2)
println(double(5)) // 10
```

#### **带 Receiver 的高阶函数（扩展函数风格）**

```kotlin
fun StringBuilder.buildString(builderAction: StringBuilder.() -> Unit): String {
    this.builderAction()  // this 指向 StringBuilder
    return this.toString()
}

// 使用
val str = StringBuilder().buildString {
    append("Hello")
    append(" Kotlin")
    append(2026)
}
```

### 5. inline 函数（性能优化）

高阶函数默认会有函数对象创建的开销，`inline` 可以消除这种开销：

```kotlin
inline fun <T> measureTime(block: () -> T): T {
    val start = System.currentTimeMillis()
    val result = block()
    println("耗时: ${System.currentTimeMillis() - start}ms")
    return result
}

// 使用
measureTime {
    // 执行耗时操作
}
```

**注意**：
- `inline` 会把 lambda 代码直接拷贝到调用处
- 可以使用 `noinline`、`crossinline` 控制行为
- 不能内联太大的函数（会增加字节码大小）

### 6. 其他高级特性

#### **匿名函数**
```kotlin
val anonymous = fun(a: Int, b: Int): Int {
    return a + b
}
```

#### **带 Receiver 的 Lambda**
```kotlin
val sum = fun Int.(other: Int): Int = this + other
println(5.sum(3)) // 8
```

#### **infix 函数**
```kotlin
infix fun Int.add(other: Int) = this + other

5 add 3  // 中缀调用
```

### 7. 实际应用场景

1. **DSL 构建**（如 kotlinx.html、Anko）
2. **异步编程**（Coroutine、Callback 转函数式）
3. **集合处理**（filter/map/reduce 等链式调用）
4. **依赖注入 / 控制反转**
5. **测试框架**（mock、verify）

### 推荐练习

```kotlin
// 实现一个通用的 retry 函数
inline fun <T> retry(
    times: Int = 3,
    block: () -> T
): T {
    var lastException: Exception? = null
    repeat(times) { attempt ->
        try {
            return block()
        } catch (e: Exception) {
            lastException = e
            println("第${attempt + 1}次尝试失败")
        }
    }
    throw lastException!!
}
```
