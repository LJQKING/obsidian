## 📊 基础命令

|命令|功能说明|
|---|---|
|`mo`|打开交互式菜单，选择要执行的操作|
|`mo --help`|显示帮助信息和所有可用命令|
|`mo --version`|显示已安装的 Mole 版本号|

有以下几种退出方式：

|按键|说明|
|---|---|
|**`q`**|快速退出（最简单）|
|**`Ctrl+C`**|强制中断（通用方式）|
|**`Esc`**|退出（某些版本支持）|

## 🧹 清理相关

|命令|功能说明|
|---|---|
|**`mo clean`**|**深度清理 + 已卸载应用残留**。清理系统缓存、临时文件、已删除应用的残留文件|
|**`mo uninstall`**|**移除已安装的应用 + 其残留**。完全卸载应用及其关联文件（比单纯删除更彻底）|
|**`mo purge`**|**清理项目构建产物**。删除 `node_modules`、`build/`、`.gradle/` 等开发文件|
|**`mo installer`**|**查找并移除安装程序文件**。清理 `.dmg`、`.pkg` 等已用过的安装文件|

## ⚡ 优化相关

|命令|功能说明|
|---|---|
|**`mo optimize`**|**刷新缓存和服务**。清理可重建缓存、修复损坏的偏好设置文件（plist）、优化系统性能|
|**`mo touchid`**|**配置 Touch ID 用于 sudo**。免去每次输入密码，用指纹认证运行高权限命令|

## 📈 监控和分析

|命令|功能说明|
|---|---|
|**`mo status`**|**实时系统健康仪表板**。显示 CPU、内存、磁盘、电池、温度、网络、进程等详细信息|
|**`mo analyze`** / `mo analyse`|**可视化磁盘占用浏览器**。交互式浏览哪些文件/文件夹占用了最多空间，可将文件移到废纸篓|

## 🛠️ 维护和配置

|命令|功能说明|
|---|---|
|**`mo completion`**|**设置 Shell 制表符补全**。在终端中输入 `mo` 后按 Tab 键自动补全子命令|
|**`mo update`**|**更新 Mole 到最新版本**|
|**`mo update --nightly`**|**更新到最新开发版本**（仅脚本安装方式）|
|**`mo remove`**|**从系统中移除 Mole**|
|**`mo history`**|**查看清理历史**|
|**`mo history --json`**|**以 JSON 格式输出清理历史**（便于编程处理）|

## 🔒 安全选项（干运行 & 保护列表）

```bash
mo clean --dry-run              # 预览即将清理的内容，不实际删除
mo uninstall --dry-run          # 预览即将卸载的应用
mo purge --dry-run              # 预览即将删除的构建产物
mo optimize --whitelist         # 管理受保护的优化规则
mo clean --whitelist            # 管理受保护的缓存
mo purge --paths                # 配置项目扫描目录
mo clean --dry-run --debug      # 预览 + 详细日志
```

## 📂 高级用法

```bash
mo analyze /Volumes             # 仅分析外置磁盘空间占用
```

## ⚠️ 安全特性

Mole 使用"安全优先"的默认设置：路径验证、受保护目录规则、保守的清理边界，以及对高风险操作进行明确确认。当风险或不确定性较高时，Mole 宁愿跳过或拒绝操作，而不是扩大删除范围。

## 💡 使用建议

1. **初次使用**：先用 `--dry-run` 预览效果
2. **系统优化**：定期运行 `mo status` 监控系统健康
3. **清理策略**：`mo clean` → `mo uninstall` → `mo optimize` 的顺序最优
4. **开发者**：`mo purge` 可快速释放大量磁盘空间

由于你用的是 Mac mini，Mole 这类工具对维持系统性能很有帮助！
